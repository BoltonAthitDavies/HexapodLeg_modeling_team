┌─ STEP 2: LAUNCH PLOTJUGGLER ────────────────────────────────────────────────┐
│                                                                              │
│  Terminal 2:                                                                 │
│  $ source install/setup.bash                                                 │
│  $ ros2 run plotjuggler plotjuggler                                          │
│                                                                              │
│  ✅ PlotJuggler window opens                                                 │
│                                                                              │
└──────────────────────────────────────────────────────────────────────────────┘

┌─ STEP 3: CONFIGURE STREAMING ───────────────────────────────────────────────┐
│                                                                              │
│  In PlotJuggler:                                                             │
│  1. Click [Streaming] button in toolbar                                     │
│  2. Select "ROS2 Topic Subscriber"                                           │
│  3. Click [Start] button                                                     │
│  4. In topic list, check:                                                    │
│     ✅ /joint_states                                                         │
│     ✅ /joint_position_controller/commands                                   │
│  5. Click [OK]                                                               │
│                                                                              │
└──────────────────────────────────────────────────────────────────────────────┘

┌─ STEP 4: ADD PLOTS ──────────────────────────────────────────────────────────┐
│                                                                              │
│  LEFT PANEL shows:                       DRAG TO PLOT AREA:                 │
│  └─ /joint_states                        ┌─────────────────────┐            │
│      ├─ header                           │                     │            │
│      ├─ name                             │   ← position[0]     │            │
│      ├─ position                         │   ← position[1]     │            │
│      │   ├─ [0]  ← rev1    ──────────►  │                     │            │
│      │   └─ [1]  ← rev_2   ──────────►  └─────────────────────┘            │
│      ├─ velocity                                                             │
│      │   ├─ [0]  ← rev1    ──────────►  ┌─────────────────────┐            │
│      │   └─ [1]  ← rev_2   ──────────►  │   ← velocity[0]     │            │
│      └─ effort                           │   ← velocity[1]     │            │
│                                          └─────────────────────┘            │
│  └─ /joint_position_controller/commands                                     │
│      └─ data                                                                 │
│          ├─ [0]  ← rev1 cmd ──────────►  ┌─────────────────────┐            │
│          └─ [1]  ← rev_2 cmd ─────────►  │   ← data[0]         │            │
│                                          │   ← data[1]         │            │
│                                          └─────────────────────┘            │
└──────────────────────────────────────────────────────────────────────────────┘

┌─ DATA REFERENCE ─────────────────────────────────────────────────────────────┐
│                                                                              │

┌─ QUICK COMMANDS ─────────────────────────────────────────────────────────────┐
│                                                                              │
│  # Check topics are publishing                                              │

│  $ ros2 topic hz /joint_position_controller/commands                        │
│                                                                              │
│  # View joint state once                                                     │
│  $ ros2 topic echo /joint_states --once                                      │
│                                                                              │
│  # List active controllers                                                   │
│  $ ros2 control list_controllers                                             │
│                                                                              │
